What the app shows

Visible time

The main screen turns remaining minutes into a large dial, so the timer can be read at a glance.

Native controls

Start, pause, reset, presets, color, sound, haptics, and notifications stay close to standard iOS patterns.

Local first

ArcTimer uses on-device settings and optional local notifications. No account, ads, tracking, or required network connection.

Core timer features

ArcTimer keeps the timer on one focused screen while still covering the controls people expect from an iOS timer.

01

Presets and dial adjustment

Start from 5, 10, 15, 20, 25, 30, 45, or 60 minutes, then drag the dial to fine-tune any minute from 1 to 60.

02

Analog-style visual timer

The remaining time is shown as a large dial with minute marks and a central time readout, available in square or circular form.

03

Native appearance controls

Use System, Light, or Dark appearance with six color themes: Blue, Mint, Lavender, Pink, Peach, and Gray.

04

Optional alerts

Choose from built-in finish sounds, enable vibration, keep haptics on for interactions, or allow local notifications when the app is away.

05

Local by default

Preferences and the rotating reflection lines are stored on device. There is no account, advertising, tracking, or required network connection.
